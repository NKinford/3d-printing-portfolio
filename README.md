# 3D Printer
A little bit about what I work with. I currently am using a ELEGOO Neptune 4 Pro, a 3D FDM printer. Now I wont lie, this is a cheap Chinese 3d printer, but it has done me wonders with what it can create, as I have
been using it for over a year. 


Now with that said, it should be no suprise that my cute little printer has gone through some maintanence. A couple months ago, the Auto Bed Leveling Sensor / Z Probe tells the print head that tells the printer where it is vertically, and somehow it magically broke. The only way to fix this issue was replacing the wire for it would be to bust it open and remove all wiring and replace it with the wiring kit that I have aqquired, it was painful, and I cut myself a few times, but in the end it still kicking and working like a charm.
