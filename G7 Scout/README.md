# G7 Scout Replica Build

![Finished Build](photo1.jpg)

## Overview

Fabricated and finished a replica of the G7 Scout from Apex Legends using a licensed third-party STL model. This project involved multi-part printing, fitment adjustments, assembly, surface finishing, and final paint work to create a display-quality prop.

## Project Details

* Model Source: Licensed third-party STL file
* Inspired By: Apex Legends
* Printer: Elegoo Neptune 4 Pro
* Material: PLA
* Print Configuration: Multi-part assembly
* Use Case: Display / Cosplay Prop

## Work Performed

* Prepared model files for segmented printing
* Oriented parts to reduce visible seam lines
* Tuned supports for detailed surfaces and overhangs
* Printed individual components and monitored print quality
* Test-fitted mating parts and corrected tolerances
* Assembled sections using adhesive and alignment methods
* Sanded surfaces and seams for smoother finish
* Primed and painted final model
* Completed final detailing and presentation

## Challenges

* Maintaining alignment across multiple large sections
* Reducing visible seams after assembly
* Preserving surface detail during sanding
* Support cleanup in narrow or detailed areas

## Solutions

* Dry-fit components before final bonding
* Used filler and progressive sanding for seam reduction
* Reprinted small sections with revised orientation when needed
* Applied controlled finishing to preserve detail lines

## Skills Demonstrated

* 3D printing workflow
* Multi-part assembly
* Troubleshooting and iteration
* Surface finishing
* Paint preparation
* Attention to detail
* Project execution
